# 🚗 Car Price Predictor — Machine Learning Model

A Machine Learning project that predicts the **MSRP (Manufacturer's Suggested Retail Price)**
of a car based on key features like Make, Engine HP, Transmission Type, and more.

---

## 📌 Project Overview

This project uses a **Random Forest Regressor** trained on a real-world car dataset
to accurately predict car prices. The model achieves an **R² score of 0.97**,
meaning it explains 97% of the variation in car prices.
